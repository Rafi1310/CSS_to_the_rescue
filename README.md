## *18 maart 2026*
### Check-out (met Jeppe)
**Wat heb ik vandaag gedaan**

Ik heb er vandaag voor gezorgd dat de switch af is. Ik heb checkboxes toegevoegd met een container query om het thema te veranderen. Er zit nu een dark mode en een 8-bit mode op. Bij de dark mode veranderen de joycons naar donkergrijs en de achtergrond naar een donkere afbeelding. Bij de 8-bit mode veranderen de letters naar een ander lettertype en de achtergrond naar een pixel afbeelding.Als je ze allebei aanvinkt is verandert die naar een donkere 8-bit met een andere achtergrond. Ook is er een 'geheime' knop die je kan inklikken waarbij het allemaal onleesbaar word. Ik heb dus veel met fonts gespeeld vandaag en met containers.
Ik heb de light dark color scheme weggehaald, omdat dit een beetje het project verpeste. Ook heb ik dus de @support weggehaald.
